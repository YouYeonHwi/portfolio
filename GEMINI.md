# PROJECT OVERVIEW

나는 AI, 자동화, 게임개발, 연구를 융합하는 개발자이자 강사이다.

이 프로젝트는 단순한 포트폴리오 사이트가 아니라,
"나를 하나의 인터랙티브한 세계관으로 표현하는 3D 웹 포트폴리오"를 목표로 한다.

웹사이트는 사용자가 탐험하는 느낌을 주어야 하며,
3D 공간 안에서 나의 프로젝트, 연구, 강의 경험을 자연스럽게 발견할 수 있어야 한다.

전체 분위기는:
- 미래지향적
- 몰입형
- 인터랙티브
- 기술적이지만 감성적인 연출
- 다크 테마 기반
- Apple + Cyberpunk + Game UI 느낌 혼합

# GAME DEVELOPMENT SECTION

## 문경새재 게임 개발 프로젝트

문경새재를 배경으로 한 게임 프로젝트를 소개하는 공간을 만든다.

핵심 방향:
- 단순 관광 소개가 아니라,
  "역사와 분위기를 탐험하는 인터랙티브 경험" 느낌

게임 분위기:
- 한국 전통 + 자연 풍경
- 안개 낀 산길
- 조선시대 감성
- 몽환적 분위기
- Unreal Engine / Indie Adventure 느낌

표현 요소:
- 3D 지형
- 문경새재 성문
- 산길
- 전통 오브젝트
- 횃불 / 안개 / 바람 효과
- 배경 새소리 및 자연 환경음

사용자가 스크롤하거나 이동하면:
- 게임 컨셉 이미지 등장
- 개발 과정 소개
- 세계관 설명
- 사용 기술 스택 표시

추가로 실제 문경새재 역사/문화 요소를 웹 검색 기반으로 참고하여
세계관 설정에 자연스럽게 녹여낸다.
